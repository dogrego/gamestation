# Space Invaders - WIP

[Play this version](https://gregoricsd.web.elte.hu/space-invaders/index.html)

- Start the game by pressig any key
- Use A / LEFT arrow key to move left and D / RIGHT arrow key to move right
- Press SPACE to shoot
